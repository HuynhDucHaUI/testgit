thẻ img là thẻ chèn ảnh vào trong file html
        thẻ img có thuộc tính src chứa đường dẫn của hình ảnh

        ** đường dẫn trong thuộc tính src có 2 loại
        loại 1 là đường dẫn tương đối
        trường hợp 1 : ảnh và file html cùng folder mẹ
        thì src="tên của ảnh.đuổi của ảnh"
        ví dụ : src="IMG_1003.JPG"
